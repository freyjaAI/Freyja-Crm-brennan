# Freyja IQ Broker CRM — Replit Setup Guide

## Files You Downloaded
1. `freyja-crm-code.zip` — The CRM application code (708 KB)
2. `data.db.gz.part-aa` — Database part 1 (190 MB)
3. `data.db.gz.part-ab` — Database part 2 (190 MB)
4. `data.db.gz.part-ac` — Database part 3 (19 MB)

## Step-by-Step Setup

### 1. Create a New Repl
- Go to [replit.com](https://replit.com) and click **+ Create Repl**
- Choose **Node.js** as the template
- Name it `freyja-crm` (or whatever you want)

### 2. Upload the Code
- In the Replit file panel, click the three dots menu → **Upload file**
- Upload `freyja-crm-code.zip`
- Open the **Shell** tab and run:
  ```bash
  unzip freyja-crm-code.zip
  mv freyja-crm/* .
  mv freyja-crm/.gitignore .
  rm -rf freyja-crm freyja-crm-code.zip
  ```

### 3. Upload the Database Parts
- Upload all three `data.db.gz.part-*` files via the file panel
- In the Shell, run:
  ```bash
  cat data.db.gz.part-aa data.db.gz.part-ab data.db.gz.part-ac > data.db.gz
  gunzip data.db.gz
  rm data.db.gz.part-*
  ```
  This reassembles and decompresses the database (~1.5 GB).

### 4. Install Dependencies
```bash
npm install
```

### 5. Configure Replit
Create a `.replit` file (or edit the existing one) with:
```
run = "NODE_ENV=production node dist/index.cjs"

[nix]
channel = "stable-24_05"

[[ports]]
localPort = 5000
externalPort = 80
```

### 6. Run It
Click the **Run** button, or in the Shell:
```bash
NODE_ENV=production node dist/index.cjs
```

The CRM will be available at your Replit URL (shown at the top of the webview).

## Notes
- The database contains **2,618,852 broker records** across all 50 states
- You need a **Replit Core or Teams plan** — the free tier doesn't have enough storage for the 1.5 GB database
- The built production files are included in `dist/` so you don't need to rebuild
- If you want to make code changes, run `npm run dev` for the dev server, then `npm run build` before deploying
